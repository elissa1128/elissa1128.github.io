ref 屬性：找到 DOM
發布於：2019-07-14 17:11:36 標籤：/ React / 訪問：

JSX 元素可以有 ref 屬性

範例：當進入畫面後，希望 input 框是 focus 的狀態

第一種方式：
官方推薦寫法，匯入 createRef 方法

將方法指定到一個變數 myInput 身上，在將變數 myInput 指定給 ref
就能用 .current 的方式找到 input

第二種方式：
傳入一個函式，需要帶入一個參數，參數是元素本身

第三種方式：
傳入一個字串
