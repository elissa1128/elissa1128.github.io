入門

上層傳下來的屬性

將 text 屬性傳給 Item 組件。如果要傳入的值是字串以外(ex.數字)，要用花括弧包起來

利用 this.props 拿到上層組件傳進來的 text

如果上層組件要傳給子組件的值夾在組件內，子組件可以用 this.props.children 取得值

進階用法

用 props 指定初始 state

用 defaultProps 指定預設 props

當子組件已經定義來自上層傳來的預設值時，而上層組件又忘了給初始值，這時子組件的 props.initCount 會是 undefined 而導致程式出錯

為了避免這種錯誤，可以用 defaultProps 指定子組件的初始預設值，有兩種寫法，一個可以在組件內指定

或是在組件外

用 propTypes 檢查 props 型別

假設今天的上下層組件是由不同人分工完成，上層組件給了一個字串 10，在子組件當作上層給個是數字直接拿來做運算，會發生字串相加的問題

這時可以在子組件用 propTypes 驗證上層傳來 props 的型別，有兩種寫法

當 props 傳來的屬性不匹配時，就會報錯
