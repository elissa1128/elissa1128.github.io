安裝

建立一個專案資料夾，並打開終端機

修改配置

開啟 _config.yml，新增以下關於 github 帳號的設定：

url: https://[github帳號].github.io

repository: http://github.com/[你的github帳號]/[你的github帳號].github.io.git

新增文章

hexo new <文章名稱>
命名規則為 kebab Case，單字與單字間使用連接符號
